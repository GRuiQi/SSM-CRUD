#### 项目简介

**SSM_CRUD**

- Create 创建

- Retrieve 查询

- Update 更新

- Delete 删除

**功能点**

1. 分页
2. 数据校验  前端+JSR303后端校验
3. ajax
4. Rest风格的URI;使用HTTP协议请求方式的动词，来表示对资源的操作GET(查询)，POST(新增)，PUT(修改)，DELETE(删除)

**技术点**

- 基础框架SSM

- 数据库 Mysql

- 前端框架  bootstrap 快速搭建简洁美观的界面

- 项目的依赖管理 Maven

- 分页 pagehelper

- 逆向工程 Mybatis Generator
